### 如何贡献

1. Pull request
    1. Fork 后，进行修改。
    2. 发起 Merge 请求。

2. 代理提交
    1. 进入[南科手册 QQ群](https://jq.qq.com/?_wv=1027&k=5D8EgDF): 905618908
    2. 提交规范编写的Word，Markdown 文件给管理员。
    3. 管理员整理上传。

### 编译版本

* nodejs版本：18
* 推荐包管理器：pnpm
  * 安装依赖：pnpm install
  * 升级依赖：pnpm up
  * 快速构建：pnpm run docs:fastbuild
  * 构建：pnpm run docs:build
  * 预览：pnpm run docs:dev
  * 清理：pnpm run docs:clean
